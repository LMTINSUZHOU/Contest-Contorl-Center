# 导入导出说明

系统支持学生、指导老师、竞赛、获奖四类 Excel 导入导出。

## 类型

| 类型 | 路径参数 | 说明 |
| --- | --- | --- |
| 学生 | `students` | 导入学生资料并创建正常账号 |
| 指导老师 | `teachers` | 导入教师资料并创建正常账号 |
| 竞赛 | `competitions` | 导入竞赛基础信息 |
| 获奖 | `awards` | 导入获奖记录 |

## 通用流程

1. 下载模板：`GET /api/admin/import-export/{type}/template`
2. 按模板填写 Excel。
3. 上传导入：`POST /api/admin/import-export/{type}/import`
4. 如有错误，根据返回的 `jobId` 下载错误报告。
5. 按条件导出：`GET /api/admin/import-export/{type}/export`

## 校验规则

- 必填字段不能为空。
- 邮箱、学号、工号、竞赛名称不能重复。
- 获奖导入时，竞赛必须已存在。
- 获奖导入时，赛道必须属于对应竞赛。
- 个人赛可直接导入，`学生学号或团队名称` 填学生学号。
- 团队赛必须先在系统中建立团队，`学生学号或团队名称` 填团队名称。
- 指导老师可填写多个工号，用英文或中文逗号分隔。
- 获奖日期格式为 `YYYY-MM-DD`，获奖地点必填。
- 错误报告包含：行号、字段、错误原因、原始值。

## 获奖模板字段

获奖导入模板列为：

`竞赛名称`、`赛道`、`竞赛别名`、`获奖主体`、`获奖等级`、`学生学号或团队名称`、`指导老师工号(逗号分隔)`、`获奖日期`、`获奖地点`、`审核状态`。

## 枚举填写

竞赛等级可填写枚举名或中文：

- `FIRST_A` / `一类A`
- `FIRST_B` / `一类B`
- `SECOND_A` / `二类A`
- `SECOND_B` / `二类B`
- `THIRD` / `三类`
- `OTHER` / `其他`

获奖等级可填写枚举名或中文：

- `GRAND_PRIZE` / `特等奖`
- `FIRST_PRIZE` / `一等奖`
- `SECOND_PRIZE` / `二等奖`
- `THIRD_PRIZE` / `三等奖`
- `EXCELLENCE` / `优秀奖`
- `EXCELLENT_ADVISOR` / `优秀指导老师奖`

获奖主体可填写：

- `STUDENT` / `个人赛`
- `TEAM` / `团队赛`
- `TEACHER` / `教师获奖`
